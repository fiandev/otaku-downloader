# otaku downloader
tools for automatically downloading anime from otakudesu

## languages
- indonesian
  [indonesian mode](https://github.com/fiandev/otaku-downloader/blob/master/ID.README.md)
- english (✓)
  [english mode](https://github.com/fiandev/otaku-downloader/blob/master/README.md)

## setup

### install nodejs

#### use termux
```
pkg update -y && pkg upgrade -y
pkg install nodejs-lts git -y
```

## installation

### install via npm

```shell
npm install otaku-downloader -g
```

### clone this repository

```shell
# clone repository
git clone https://github.com/fiandev/otaku-downloader

# enter your local repository folder
cd otaku-downloader

# installing required modules
npm install -g
```

## how to usage

### call npm executable command

```shell
npx otaku
```
